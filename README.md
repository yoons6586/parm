# GPS 기반 알림 서비스

위치 기반 푸시 알림 서비스 애플리케이션입니다.

## 기술 스택

### Frontend
- Flutter
- Google Maps SDK
- Firebase Cloud Messaging (FCM)

### Backend
- Kotlin
- Spring Boot
- Spring Data JPA
- PostgreSQL (위치 데이터 저장)
- Redis (실시간 위치 캐싱)

## 주요 기능
- 사용자 위치 기반 실시간 알림
- Google Maps를 통한 위치 시각화
- FCM을 통한 푸시 알림 전송
- 위치 기반 알림 서비스
